# Moontime Font

Place your Moontime font files in this directory.

## Required Files

Add at least one of the following font file formats (preferred order):
- `Moontime-Regular.woff2` (most efficient, preferred)
- `Moontime-Regular.woff` (good fallback)
- `Moontime-Regular.ttf` (widely compatible)
- `Moontime-Regular.otf` (OpenType format)

## How to Obtain Moontime Font

Moontime is a commercial script font. You can purchase and download it from:
- [Creative Market](https://creativemarket.com/Supfonts/2299867-Moontime-Script)
- [Dfonts](https://www.dfonts.org/fonts/moontime-script/)

After downloading, rename the font file to match one of the names above and place it in this directory.
