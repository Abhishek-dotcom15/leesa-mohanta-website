Resource file drop zones for the Resources page.

Supported folders:
- `public/resources/odissi-natyashastra`
- `public/resources/handlooms-crafts`
- `public/resources/arts-entrepreneurship`
- `public/resources/expanding-thought`

Supported file types:
- Images: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`, `.avif`, `.bmp`, `.svg`
- Presentations: `.ppt`, `.pptx`, `.odp`, `.key`
- Documents: `.pdf`, `.doc`, `.docx`
- Videos: `.mp4`, `.webm`, `.mov`, `.m4v`

You can add any number of files in each folder. The page reads these folders dynamically.
